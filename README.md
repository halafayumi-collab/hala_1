# Integer Statistics Tool / أداة إحصائيات الأرقام الصحيحة
A Java-based application that processes a sequence of integers. It performs statistical calculations and provieds a summray of the entered data.

**Key Features :**
* Calculates the total sum of entered integers.
* Counts the numbers of positive and negative values.
* Computes the average value of the inputs.
* Handles user input efficiently using Java logic.

  ---

  برنامج مكتوب بلغة جافا يقوم بمعالجة مجموعة من الأرقام الصحيحة التي يدخلها المستخدم ، و يقوم بإجراء عمليات إحصائية عليها وتقديم ملخص للبيانات.

  **المميزات الأساسية :**
  * حساب المجموع الكلي للأرقام المدخلة.
  * حساب عدد الأرقام الموجبة و السالبة بشكل منفصل.
  * حساب المتوسط الحسابي للقيم.
  * التعامل مع مدخلات المستخدم بكفاءة بإستخدام منطق البرمجة في جافا.
